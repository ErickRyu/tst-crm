# 업데이트 소식

## [0.4.0] - 2026-03-10

### 새 기능
- **로그인 및 권한 관리**: 이메일/비밀번호로 로그인할 수 있으며, 관리자/상담사/병원스태프 3가지 역할별로 접근 가능한 메뉴가 구분됩니다
- **비밀번호 변경**: 첫 로그인 시 비밀번호를 변경해야 하며, 원할 때 언제든 비밀번호를 변경할 수 있습니다
- **사용자 관리 화면**: 관리자가 사용자 목록을 조회하고 역할을 변경할 수 있습니다
- **병원스태프 전용 캘린더**: 병원스태프는 예약 캘린더를 읽기 전용으로 확인할 수 있습니다
- **텔레그램 다중 수신자**: 텔레그램 알림을 여러 명에게 동시에 발송할 수 있습니다
- **연락처 저장 개선**: 저장되는 연락처에 치과 정보(주소, 전화번호)가 함께 포함됩니다

### 개선
- **텔레그램 알림 안정성**: 네트워크 오류 시 자동으로 재시도하여 알림이 더 안정적으로 발송됩니다
- **텔레그램 알림 전화번호**: 알림에 전화번호가 원본 그대로 표시됩니다
- **날짜 필터 정확도**: 날짜 기준이 한국 시간(KST)으로 정확하게 적용됩니다

### 버그 수정
- 모바일에서 상세 화면이 좌우로 흔들리는 현상을 수정했습니다

## [0.3.0] - 2026-03-04

### 새 기능
- **텔레그램 알림 연동**: 텔레그램 봇을 연결하면 새 상담이 들어올 때 바로 알림을 받을 수 있습니다. 설정 화면에서 단계별로 쉽게 연결할 수 있습니다
- **SMS 미리보기**: 문자 발송 전 실제 핸드폰 화면처럼 미리 확인할 수 있습니다

### 개선
- **활동 이력 보기**: 긴 내용은 접어서 보여주고, 필요할 때 펼쳐볼 수 있습니다
- **메뉴 구성 개선**: 사이드바와 상세 화면이 더 깔끔하게 정리되었습니다

### 버그 수정
- 목록이 자동 갱신될 때 카드가 잠깐 깜빡이는 현상을 수정했습니다

## [0.2.0] - 2026-03-04

### 새 기능
- **SMS 템플릿 관리**: 기본/커스텀 템플릿을 관리하고 자동발송을 설정할 수 있습니다
- **SMS 템플릿 화면 개선**: 아이콘으로 한눈에 구분하고, 글자 수를 자동으로 계산해줍니다
- **새 상담 상태 추가**: 추후 통화희망, 추가상담거부, 블랙리스트, 중복 상태가 추가되었습니다
- **활동 이력**: 상태 변경, 메모, SMS 등 모든 활동 내역을 타임라인으로 확인할 수 있습니다
- **연락처 저장**: 스마트폰에 바로 연락처를 저장할 수 있습니다
- **엑셀 다운로드**: 환자 정보를 엑셀 파일로 내려받을 수 있습니다
- **날짜 필터링**: 유입일 기준으로 날짜 범위를 지정하여 조회할 수 있습니다
- **정렬 기능**: 유입날짜 기준 오름차순/내림차순 정렬이 가능합니다
- **리스트 뷰 페이지네이션**: 데이터가 많아도 빠르게 조회할 수 있습니다
- **캘린더 예약 시간 표시**: 캘린더에 예약 시간이 표시되고 시간순으로 정렬됩니다
- **업데이트 소식 페이지**: 앱 내에서 업데이트 내역을 확인할 수 있습니다
- **앱 버전 표시**: 현재 앱 버전을 화면에서 확인할 수 있습니다

### 개선
- **UI 디자인 통일**: 전체적으로 일관된 디자인으로 개선했습니다
- **헤더 레이아웃 개편**: 상단 메뉴와 필터를 분리하여 더 넓게 사용할 수 있습니다
- **상세 화면 정리**: 상담메모를 최상단으로 이동하고, 정보를 한눈에 파악할 수 있도록 정리했습니다
- **리스트 뷰 개선**: 순번과 유입일 컬럼을 추가하여 가독성을 높였습니다
- **칸반 기간 필터**: 완료 항목을 기간별로 필터링할 수 있습니다
- **랜딩 페이지 리디자인**: 새로운 디자인의 랜딩 페이지로 변경했습니다

### 버그 수정
- 모바일에서 상단 메뉴가 겹치는 문제를 수정했습니다
- 일부 화면에서 발생하던 오류를 수정했습니다
- 리스트 뷰 예약 컬럼이 과도하게 넓었던 문제를 수정했습니다

## [0.1.0] - 2025-02-26

### 새 기능
- **CRM 대시보드**: 칸반 보드, 리스트, 캘린더 3가지 뷰 모드를 제공합니다
- **모바일 지원**: 스마트폰에서도 편리하게 사용할 수 있는 반응형 레이아웃
- **실시간 갱신**: 30초마다 자동으로 최신 데이터를 불러옵니다
- **메모 기능**: 칸반 카드에서 바로 메모를 입력하고 미리보기할 수 있습니다
- **메모 관리**: 메모 작성, 수정, 삭제 및 이전 버전 확인이 가능합니다
- **SMS 발송**: Aligo 연동으로 SMS를 보내고, 상태별 추천 템플릿을 제공합니다
- **리드 상세 보기**: 고객 정보를 드로어 패널에서 한눈에 확인할 수 있습니다
- **전화번호 연결**: 전화번호 터치로 바로 전화 연결 및 복사가 가능합니다
- **알림 배지**: 24시간 이상 미응답 또는 노쇼 고객을 자동으로 표시합니다
- **충돌 방지**: 여러 상담원이 동시에 수정해도 안전하게 처리됩니다

### 개선
- **카드 디자인**: 나이, 성별, 채널, 시간 배지로 한눈에 정보를 파악할 수 있습니다
- **상태 변경**: 그리드 버튼 대신 드롭다운으로 더 쉽게 상태를 변경합니다
- **일정 표시 간소화**: 팔로업 일정을 제거하고 예약 일정만 표시합니다
- **카드 정리**: 칸반 카드에서 상담원 배정 표시를 제거하여 깔끔하게 정리했습니다
- **세탁 버튼 이동**: 세탁 버튼을 드로어 메모 섹션으로 옮겨 접근성을 개선했습니다

### 버그 수정
- **새로고침 문제**: 데이터를 갱신할 때 전체 페이지가 새로고침되던 문제를 해결했습니다
- **모바일 드롭다운**: 모바일에서 상담원 드롭다운 위치가 잘못 표시되던 문제를 수정했습니다
- **모바일 드롭다운 교체**: 모바일 상담원 드롭다운을 커스텀 드롭다운으로 교체했습니다
- **불필요한 UI 제거**: 칸반 카드에서 사용하지 않는 모바일 상태 드롭다운을 제거했습니다
