# 업데이트 소식

## [0.2.0] - 2026-03-04

### 새 기능
- **SMS 템플릿 관리**: DB 기반 템플릿 관리, 기본/커스텀 구분, 자동발송 설정을 지원합니다
- **SMS 템플릿 UI 개선**: 아이콘 그리드, 자동 바이트 계산, 순서 정렬 등 편리한 관리 화면을 제공합니다
- **새 CRM 상태**: 추후 통화희망, 추가상담거부, 블랙리스트, 중복 상태가 추가되었습니다
- **활동 이력 타임라인**: 드로어에서 리드의 상태 변경, 메모, SMS 등 모든 활동 이력을 확인할 수 있습니다
- **모바일 연락처 저장**: vCard 다운로드로 스마트폰에 바로 연락처를 저장할 수 있습니다
- **엑셀 다운로드**: 환자 정보를 엑셀 파일로 내려받을 수 있습니다
- **날짜 필터링**: 리드 목록에서 유입일 기준으로 날짜 범위를 지정하여 조회할 수 있습니다
- **정렬 기능**: 유입날짜 기준 오름차순/내림차순 정렬 토글을 제공합니다
- **리스트 뷰 페이지네이션**: 서버 사이드 페이지네이션으로 대량 데이터도 빠르게 조회합니다
- **캘린더 예약 시간 표시**: 캘린더뷰 이벤트에 예약 시간이 표시되고 시간순으로 정렬됩니다
- **업데이트 타임라인**: 앱 내에서 업데이트 내역을 확인할 수 있는 페이지가 추가되었습니다
- **앱 버전 표시**: 현재 앱 버전을 화면에서 확인할 수 있습니다

### 개선
- **ShadCN UI 마이그레이션**: 기존 커스텀 UI를 ShadCN 컴포넌트로 전환하여 일관성을 높였습니다
- **헤더 레이아웃 개편**: 네비게이션 바를 프라이머리 바와 접이식 필터 패널로 분리했습니다
- **드로어 영역 정리**: 상담메모를 최상단으로 이동하고, 정보를 통합하여 한눈에 파악할 수 있습니다
- **리스트 뷰 개선**: 순번 넘버링, 유입일 별도 컬럼 분리로 가독성을 높였습니다
- **칸반 기간 필터**: 완료 항목 체크박스를 제거하고 기간 필터로 대체했습니다
- **랜딩 페이지 리디자인**: Attio 스타일의 프로덕션급 랜딩 페이지로 새롭게 디자인했습니다
- **Aligo API 직접 호출**: aligoapi 패키지를 제거하고 fetch 기반 직접 호출로 전환했습니다

### 버그 수정
- **모바일 헤더 겹침**: 모바일에서 헤더 네비게이션 바가 겹치는 문제를 수정했습니다
- **careTag 에러**: careTag가 undefined일 때 발생하는 렌더링 에러를 방지했습니다
- **마이그레이션 스크립트**: migrate-0006 SQL 파싱 버그를 수정했습니다
- **리스트뷰 예약 컬럼**: 예약 일정 컬럼 영역이 과도하게 넓었던 문제를 수정했습니다

### 테스트
- **CRM 테스트 코드**: 주요 기능에 대한 64개 테스트를 추가했습니다

## [0.1.0] - 2025-02-26

### 새 기능
- **CRM 대시보드**: 칸반 보드, 리스트, 캘린더 3가지 뷰 모드를 제공합니다
- **모바일 지원**: 스마트폰에서도 편리하게 사용할 수 있는 반응형 레이아웃
- **실시간 갱신**: 30초마다 자동으로 최신 데이터를 불러옵니다
- **메모 기능**: 칸반 카드에서 바로 메모를 입력하고 미리보기할 수 있습니다
- **메모 관리**: 메모 작성, 수정, 삭제 및 이전 버전 확인이 가능합니다
- **SMS 발송**: Aligo 연동으로 SMS를 보내고, 상태별 추천 템플릿을 제공합니다
- **리드 상세 보기**: 고객 정보를 드로어 패널에서 한눈에 확인할 수 있습니다
- **전화번호 연결**: 전화번호 터치로 바로 전화 연결 및 복사가 가능합니다
- **알림 배지**: 24시간 이상 미응답 또는 노쇼 고객을 자동으로 표시합니다
- **충돌 방지**: 여러 상담원이 동시에 수정해도 안전하게 처리됩니다

### 개선
- **카드 디자인**: 나이, 성별, 채널, 시간 배지로 한눈에 정보를 파악할 수 있습니다
- **상태 변경**: 그리드 버튼 대신 드롭다운으로 더 쉽게 상태를 변경합니다
- **일정 표시 간소화**: 팔로업 일정을 제거하고 예약 일정만 표시합니다
- **카드 정리**: 칸반 카드에서 상담원 배정 표시를 제거하여 깔끔하게 정리했습니다
- **세탁 버튼 이동**: 세탁 버튼을 드로어 메모 섹션으로 옮겨 접근성을 개선했습니다

### 버그 수정
- **새로고침 문제**: 데이터를 갱신할 때 전체 페이지가 새로고침되던 문제를 해결했습니다
- **모바일 드롭다운**: 모바일에서 상담원 드롭다운 위치가 잘못 표시되던 문제를 수정했습니다
- **모바일 드롭다운 교체**: 모바일 상담원 드롭다운을 커스텀 드롭다운으로 교체했습니다
- **불필요한 UI 제거**: 칸반 카드에서 사용하지 않는 모바일 상태 드롭다운을 제거했습니다
