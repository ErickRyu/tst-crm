# 업데이트 소식

## [0.1.0] - 2025-02-26

### 새 기능
- **CRM 대시보드**: 칸반 보드, 리스트, 캘린더 3가지 뷰 모드를 제공합니다
- **모바일 지원**: 스마트폰에서도 편리하게 사용할 수 있는 반응형 레이아웃
- **실시간 갱신**: 30초마다 자동으로 최신 데이터를 불러옵니다
- **메모 기능**: 칸반 카드에서 바로 메모를 입력하고 미리보기할 수 있습니다
- **메모 관리**: 메모 작성, 수정, 삭제 및 이전 버전 확인이 가능합니다
- **SMS 발송**: Aligo 연동으로 SMS를 보내고, 상태별 추천 템플릿을 제공합니다
- **리드 상세 보기**: 고객 정보를 드로어 패널에서 한눈에 확인할 수 있습니다
- **전화번호 연결**: 전화번호 터치로 바로 전화 연결 및 복사가 가능합니다
- **알림 배지**: 24시간 이상 미응답 또는 노쇼 고객을 자동으로 표시합니다
- **충돌 방지**: 여러 상담원이 동시에 수정해도 안전하게 처리됩니다

### 개선
- **카드 디자인**: 나이, 성별, 채널, 시간 배지로 한눈에 정보를 파악할 수 있습니다
- **상태 변경**: 그리드 버튼 대신 드롭다운으로 더 쉽게 상태를 변경합니다
- **일정 표시 간소화**: 팔로업 일정을 제거하고 예약 일정만 표시합니다
- **카드 정리**: 칸반 카드에서 상담원 배정 표시를 제거하여 깔끔하게 정리했습니다
- **세탁 버튼 이동**: 세탁 버튼을 드로어 메모 섹션으로 옮겨 접근성을 개선했습니다

### 버그 수정
- **새로고침 문제**: 데이터를 갱신할 때 전체 페이지가 새로고침되던 문제를 해결했습니다
- **모바일 드롭다운**: 모바일에서 상담원 드롭다운 위치가 잘못 표시되던 문제를 수정했습니다
- **모바일 드롭다운 교체**: 모바일 상담원 드롭다운을 커스텀 드롭다운으로 교체했습니다
- **불필요한 UI 제거**: 칸반 카드에서 사용하지 않는 모바일 상태 드롭다운을 제거했습니다
