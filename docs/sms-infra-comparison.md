## 3. 업체별 상세 분석

### Aligo (알리고) — 비용 최적화

**장점:**
- 국내 최저가 수준 (SMS 8.4원, 알림톡 6.5원)
- 080 수신거부 서비스 무료 제공 (광고 문자 법적 필수)
- 알림톡 실패 시 자동 SMS 대체 발송
- 충전금 5년 유효 — 소량 발송 업체에 유리
- 실패 건 자동 환불

**단점:**
- API 문서가 다소 기본적 (한국어 only)
- SDK 품질이 SOLAPI 대비 낮음 (TypeScript 미지원)
- 최소 충전금 50,000원

**적합 대상:** 비용이 최우선이고 발송량이 많은 경우

---

### SOLAPI (CoolSMS) — 개발자 경험 최적화

**장점:**
- 공식 `solapi` npm 패키지 — Next.js 연동 가장 깔끔
- TypeScript 지원, 웹훅, 알림톡 템플릿 API 완비
- REST API v4, 한영 문서 모두 제공
- 월 기본료 0원, 최소 충전 없음
- 볼륨 할인 시 단가 경쟁력 확보 가능 (5M+ 시 SMS 8.4원)

**단점:**
- 기본 단가가 Aligo 대비 약 2배
- 볼륨 할인 혜택은 대량 발송 시에만 의미 있음

**적합 대상:** 개발 생산성과 유지보수 용이성이 우선인 경우

---

### NHN Cloud — 엔터프라이즈

**장점:**
- 알림톡 단가 최저 (5~7.9원)
- 엔터프라이즈급 안정성과 SLA
- RCS 지원, 통계 대시보드 제공

**단점:**
- 공식 Node.js SDK 없음 (3rd party만 존재)
- 가격 협상 필요, 소규모 업체에 비효율적

**적합 대상:** 대규모 엔터프라이즈 환경

---

### Twilio — 비추천

**비추천 사유:**
- `[국제발신]` 태그 강제 — 환자에게 스팸으로 인식
- 카카오 알림톡 미지원
- LMS/MMS 미지원
- 단가 6~8배 비쌈 (SMS ~66원)

## 4. 치과 CRM 발송 채널 구성안

```
예약 리마인더 / 치료 안내  →  카카오 알림톡 (정보성, 6.5~13원)
알림톡 실패 시             →  SMS 자동 대체 발송 (8.4~18원)
프로모션 (할인/이벤트)      →  LMS + (광고) 표기 + 080 수신거부
부재중 안내                →  SMS 단문 (8.4~18원)
```

## 5. 법적 필수 사항

### 발신번호 사전등록제 (전기통신사업법 제84조)

모든 발신번호는 사전 등록 필수. 필요 서류:
- 사업자등록증
- 통신서비스이용증명원 (1개월 이내 발급)
- 재직증명서 (직원 명의 등록 시)

### 광고 문자 규정 (정보통신망법 제50조)

| 규정 | 내용 |
|------|------|
| 사전 동의 | 수신자 opt-in 동의 필수 |
| 광고 표시 | 메시지 앞에 `(광고)` + 업체명 |
| 수신 거부 | 무료 080 수신거부번호 표시 의무 |
| 야간 발송 | 21:00~08:00 광고 문자 금지 (별도 야간 동의 시 가능) |
| 처리 의무 | 수신 거부 요청 즉시 반영 |

**위반 시 벌칙:** 최대 징역 3년 또는 벌금 3,000만원

### 치과 CRM 특이사항

- **정보성 메시지** (예약 리마인더, 진료 안내): `(광고)` 표기 불필요, 알림톡 사용 가능
- **광고성 메시지** (화이트닝 할인, 임플란트 이벤트): 모든 광고 규정 준수 필수
- **환자 동의**: 진료 신청서에서 문자 수신 동의 수집 권장

## 6. 최종 선정: Aligo (알리고)

**선정 근거:**
1. 국내 최저가 — SMS 8.4원, 알림톡 6.5원
2. 080 수신거부 서비스 무료 제공 (광고 문자 법적 필수)
3. 알림톡 실패 시 자동 SMS 대체 발송 지원
4. 충전금 5년 유효 — 소규모 발송에 유리
5. 발송 실패 건 자동 환불

**비용 구조:**
- 최소 충전: 50,000원 (충전 옵션: 5만/10만/30만/50만/100만원)
- 월 기본료: 없음
- 가입 무료 포인트: 없음
- 무료 발송 건수: 없음

**테스트 모드:**
- API 파라미터 `testmode_yn=Y` 설정 시 과금 없이 연동 테스트 가능
- 실제 문자는 발송되지 않으며 조회 시 "취소완료"로 표시

**5만원 충전 시 발송 가능 건수:**
| 채널 | 단가 | 발송 가능 건수 |
|------|------|---------------|
| 알림톡 | 6.5원 | ~7,690건 |
| SMS | 8.4원 | ~5,950건 |
| LMS | 25.9원 | ~1,930건 |

**API 연동 정보:**
- 인증: API Key + User ID
- Node.js SDK: `aligoapi` (npm)
- 테스트: `testmode_yn=Y`로 무과금 테스트 후 실발송 전환
- 문서: https://smartsms.aligo.in/admin/api/spec.html
