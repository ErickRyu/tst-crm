# Changelog

이 프로젝트의 모든 주요 변경사항을 기록합니다.
[Keep a Changelog](https://keepachangelog.com/ko/1.1.0/) 형식을 따릅니다.

## [0.2.0] - 2026-03-04

### Added
- SMS 템플릿 DB 관리 및 자동발송 설정 구현
- SMS 템플릿 관리 UI/UX 개선 (아이콘 그리드, 자동 바이트 계산, 순서 정렬)
- SMS 템플릿 기본/커스텀 구분 및 기본 템플릿 토글 기능
- 새 CRM 상태 UI 적용 (추후 통화희망, 추가상담거부, 블랙리스트, 중복)
- 드로어 활동 이력(History) 타임라인 기능
- 모바일 연락처 저장 기능 (vCard 다운로드)
- 캘린더뷰 이벤트에 예약 시간 표시 및 시간순 정렬
- 리스트 뷰 서버 사이드 페이지네이션 및 칸반 DONE 기간 제한
- 리스트 뷰 유입일 별도 컬럼 분리
- 리스트 뷰 순번(No.) 넘버링
- 리드 목록 날짜 필터링 기능
- 환자 정보 엑셀 다운로드 기능
- 유입날짜 기준 오름차순/내림차순 정렬 토글
- 메모 API upsert 방식 전환 (최신 1개만 조회/업데이트)
- 업데이트 타임라인 페이지
- 랜딩 페이지 Attio 스타일 프로덕션 리디자인
- 앱 버전 환경변수 노출 (NEXT_PUBLIC_APP_VERSION)
- CRM 주요 기능 테스트 코드 64개

### Changed
- ShadCN UI 마이그레이션 및 모노리스 컴포넌트 분리
- 헤더 네비게이션 바를 프라이머리 바 + 접이식 필터 패널로 분리
- 드로어 섹션 순서 변경 (상담메모 최상단 이동)
- 드로어 영역 정리 (정보 통합, 상담원 제거, 메모 디폴트 텍스트)
- 완료 항목 포함 체크박스 제거, 칸반 기간 필터로 대체
- aligoapi 패키지 제거, Aligo REST API fetch 직접 호출 전환
- SMS 발송 시 자동 시스템 메모 기록 제거
- 리드 목록 기본 정렬을 유입일 내림차순으로 변경

### Fixed
- 모바일 헤더 네비게이션 바 겹침 문제
- careTag undefined 에러 방지 조건부 렌더링
- migrate-0006 스크립트 SQL 파싱 버그
- 리스트뷰 예약 일정 컬럼 영역 축소

## [0.1.0] - 2025-02-26

### Added
- 칸반 보드, 리스트, 캘린더 3가지 뷰 모드 CRM 대시보드
- 모바일 반응형 레이아웃
- 30초 폴링 기반 실시간 데이터 갱신
- 칸반 카드에 메모 인라인 입력 및 미리보기 표시
- 메모 CRUD (생성/수정/삭제) 및 버전 관리
- SMS 발송 기능 (Aligo 연동) 및 상태별 추천 템플릿
- 리드 상세 드로어 (로딩/에러 상태 포함)
- 전화번호 tel: 링크 및 복사 버튼
- 24시간 이상 미응답 / 노쇼 알림 배지
- 낙관적 잠금(Optimistic Locking) 기반 충돌 처리
- 토스트 알림 및 로딩 오버레이 공유 컨텍스트

### Changed
- 칸반 카드 레이아웃 개선 (나이/성별/채널/시간 배지)
- 상태 변경 UI를 그리드 버튼에서 드롭다운으로 변경
- 팔로업 일정 UI 제거, 예약 일정만 유지
- 칸반 카드에서 상담원 배정 표시 제거
- 세탁 버튼을 칸반 카드에서 드로어 메모 섹션으로 이동

### Fixed
- 데이터 갱신 시 전체 페이지 새로고침 방지
- 모바일 상담원 드롭다운 위치 이상 수정
- 모바일 상담원 드롭다운을 커스텀 드롭다운으로 교체
- 칸반 카드에서 불필요한 모바일 상태 드롭다운 제거
